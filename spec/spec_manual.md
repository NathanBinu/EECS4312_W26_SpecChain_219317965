# Requirement ID: RM1

- Description: [The system shall allow users to complete a structured mood check-in and save the result as a mood entry.]
- Source Persona: [PM-01]
- Traceability: [Derived from review group G1]
- Acceptance Criteria: [Given the user starts a mood check-in When the user completes and submits the check-in Then the system must save the mood entry successfully.]

# Requirement ID: RM2

- Description: [The system shall allow users to view previously saved mood entries in chronological order.]
- Source Persona: [PM-01]
- Traceability: [Derived from review group G1]
- Acceptance Criteria: [Given the user opens mood history When the entries are displayed Then the system must show entries ordered by date.]
