# Requirement ID: RH1

- Description: [The system shall allow users to complete a structured mood check-in and save the result as a mood entry.]
- Source Persona: [PH1]
- Traceability: [Derived from review group G1]
- Acceptance Criteria: [Given the user starts a mood check-in When the user completes and submits the check-in Then the system shall save the mood entry successfully.]

- Traceability: [Derived from review group G1]
